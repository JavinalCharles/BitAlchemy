Called once after the entity owner is added to the main loop. 

This \mbox{\hyperlink{classba_1_1BoxCollider_a1febea30247963e739876559c122a6f2}{awake()}} does nothing, but subtypes should override this to ensure data is instantiated properly and the owner entity has the prerequisites needed to use that component. i.\+e. \mbox{\hyperlink{classba_1_1Animation}{Animation}} component needed \mbox{\hyperlink{classba_1_1Sprite}{Sprite}} component to work.
