\doxylink{classtinyxml2_1_1XMLNode}{XMLNode} is a base class for every object that is in the XML Document Object Model (DOM), except XMLAttributes. Nodes have siblings, a parent, and children which can be navigated. A node is always in a \doxylink{classtinyxml2_1_1XMLDocument}{XMLDocument}. The type of a \doxylink{classtinyxml2_1_1XMLNode}{XMLNode} can be queried, and it can be cast to its more defined type.

A \doxylink{classtinyxml2_1_1XMLDocument}{XMLDocument} allocates memory for all its Nodes. When the \doxylink{classtinyxml2_1_1XMLDocument}{XMLDocument} gets deleted, all its Nodes will also be deleted.

\begin{DoxyVerb}A Document can contain: Element (container or leaf)
                        Comment (leaf)
                        Unknown (leaf)
                        Declaration( leaf )

An Element can contain: Element (container or leaf)
                        Text    (leaf)
                        Attributes (not on tree)
                        Comment (leaf)
                        Unknown (leaf)\end{DoxyVerb}

Accept a hierarchical visit of the nodes in the Tiny\+XML-\/2 DOM. Every node in the XML tree will be conditionally visited and the host will be called back via the \doxylink{classtinyxml2_1_1XMLVisitor}{XMLVisitor} interface.
